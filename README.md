
Project Title:

Plant Nursery Management System Using Java, JDBC and MySQL

Introduction:

The Plant Nursery Management System is a console-based Java application developed to manage
plant records efficiently.
The system uses Java for programming, JDBC for database connectivity, and MySQL for storing
data.
It helps to maintain plant details such as name, type, price, and quantity in an organized manner.

Objectives:

• To develop a Java-based application using JDBC
• To perform database operations like insert, update, delete, and search
• To reduce manual record keeping
• To understand Java and MySQL connectivity

Scope of the Project:

This project is useful for small and medium plant nurseries.
It provides an easy way to store and manage plant information using a database, improving
efficiency and accuracy.

Modules:

Admin Module:

• Add plant details
• Update plant information
• Delete plant records
• View all plants
User Module:

• View available plants
• Search plant by name

Functional Requirements:
• Add plant records
• Display plant details
• Update price and quantity
• Delete plant data
• Search plant information

Technologies Used:

• Programming Language: Java
• Database: MySQL
• Connectivity: JDBC

Conclusion:

The Plant Nursery Management System provides a simple and efficient solution for managing
nursery plant records.
It demonstrates the use of Java and JDBC to interact with a MySQL database and perform CRUD
operations.
